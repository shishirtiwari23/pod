# Skill: Frontend Expert

## Trigger
Activated when the current task involves heavy CSS, layout changes, premium animations, or significant UI overhauls.

## Objective
Ensure the frontend matches a high-end, premium aesthetic, avoiding typical developer placeholder designs. Make use of glassmorphism, fluid physics-based animations, and responsive layouts.

## Instructions
1. Analyze the core aesthetic of the application (e.g. dark mode, gradients).
2. Avoid generic padding and colors; use specific HSL tailored colors and polished spacing.
3. If constructing DOM interaction, ensure real-time visual feedback exists for the user (like typing indicators, status orbs, etc).

## Safety Rules
- Do not introduce massive external component libraries unless approved (Tailwind over bootstrap etc).
- Keep all modifications mobile-responsive by default.
